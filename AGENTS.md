# Instructions pour l'IA (OpenCode)

## Avant chaque git push

1. **Vérifier les noms de fichiers** — exécuter `python3 .opencode/scripts/check_windows_filenames.py`
2. Si des fichiers sont renommés → commit séparé `"Renomme fichiers incompatibles Windows"`
3. Puis push

## Règles du projet

- Langue : Français (code, docs, messages de commit)
- Base de données : SQLite
- Backend : FastAPI
- Frontend : Streamlit
- Scraping : DrissionPage
